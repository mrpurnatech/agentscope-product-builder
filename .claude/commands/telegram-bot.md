Start and manage the Telegram bot.

Steps:
1. Check `.env` has `TELEGRAM_BOT_TOKEN` and `TELEGRAM_USER_ID` set (non-empty, non-placeholder)
2. Check `ANTHROPIC_API_KEY` is set
3. Activate venv: `source .venv/bin/activate`
4. Run `python telegram_bot.py`
5. Report the bot status and authorized user ID
